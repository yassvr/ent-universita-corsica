# ENT Università di Corsica

Projet pédagogique de développement web consistant à reproduire
les principales pages de l’ENT de l’Università di Corsica.

## Pages réalisées
- Page d’authentification (index.html)
- Tableau de bord – Cours en ligne
- Page de détail d’un cours

## Technologies utilisées
- HTML5
- CSS3 (Flexbox, Grid)
